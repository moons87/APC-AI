# DESIGN.md — Урок·Фабрика (бренд-шпаргалка для видео)

Продукт: ИИ-платформа для преподавателя. Из одной темы — план + презентация +
обучающее видео + аудиоподкаст за ~10 минут. Плюс анализ проведённого урока.
Тон: уверенный, тёплый, «продуктовый» (как Vercel/Stripe), не корпоративно-сухой.

## Цвета
- Фон-канвас: `#eef1f6` (светлый сине-серый)
- Поверхность/карточка: `#ffffff`
- Тёмный фон (для hook/CTA): `#14161c` (чернильный)
- Текст: `#14161c` (ink) / вторичный `#3a3f4d` / приглушённый `#767c8c`
- Граница: `#e7e9f0`
- **Фирменный градиент** (главный акцент): `linear-gradient(135deg, #06b6d4 0%, #6366f1 55%, #8b5cf6 100%)` (cyan → indigo → violet)
- Праймери (заливка/фокус): `#4f46e5`
- Данные/семантика: преподаватель `#6366f1`, студенты `#10b981`, открытые вопросы `#0ea5e9`, закрытые `#f59e0b`; успех `#059669`, ошибка `#e11d48`

## Типографика
- **Display (заголовки, крупные числа): `Unbounded`** (вес 600–800), letter-spacing −0.02…−0.03em. Характерный геометрический шрифт.
- **UI/текст: `Manrope`** (400–800).
- Крупные заголовки часто залиты фирменным градиентом (background-clip: text).

## Компоненты / стиль
- Карточки: белые, скругление 14–26px, мягкие слоистые тени, тонкая граница, верхняя акцентная полоска-градиент у метрик.
- Кнопки: градиентная заливка, скругление 12–14px, тень с indigo-свечением, стрелка `→`.
- Бейджи/«eyebrow»: пилюли с мягким градиент-фоном и indigo-текстом.
- Глубина через мягкие тени и «aurora»-пятна (cyan/indigo/violet blur) на фоне + тонкая сетка.

## Do
- Доминирует светлый фон; градиент — точечный акцент (заголовки, полоски, кнопки).
- Крупная типографика Unbounded для «вау»-моментов и чисел (10, 4, 1).
- Скруглённые карточки с глубиной, плавные cubic-bezier(.2,.7,.2,1) переходы.

## Don't
- Не делать «фиолетовый градиент на белом» как клише по всей площади — градиент дозированно.
- Не использовать Inter/Roboto/Arial/системные шрифты — только Unbounded + Manrope.
- Не перегружать: воздух важнее плотности.
